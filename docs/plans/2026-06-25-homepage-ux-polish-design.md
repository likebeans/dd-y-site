# Homepage UX Polish Design

## Goal

Make the homepage feel more structured and alive without losing the restrained technical-editorial tone.

## Changes

- Add numbered section bands so scrolling reveals clear chapter boundaries.
- Add a subtle cursor-driven background field based on engineering guide lines.
- Add a lightweight `ZH / EN` language control with persistent preference.
- Add a visual image slot for project screenshots, architecture diagrams, or representative photos.
- Replace text arrows with SVG icons for more stable rendering.

## Constraints

- Keep the paper, ink, grid, and rare-lime visual language.
- Avoid decorative blobs, heavy gradients, or marketing-style cards.
- Preserve mobile readability and no horizontal overflow.
- Treat multilingual support as a homepage-level framework first; long-form article translation can be added later.
